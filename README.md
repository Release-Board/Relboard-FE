# RelBoard Frontend

RelBoard 공개 사이트 프론트엔드 (Next.js App Router).

## Stack
- Next.js + TypeScript
- styled-components
- TanStack Query
- Jest + React Testing Library
- Playwright (E2E)

## Scripts
```bash
npm run dev
npm run build
npm run start
npm run lint
npm run test
npm run test:coverage
npm run playwright
```

## Notes
- Playwright 최초 실행 전: `npx playwright install`
